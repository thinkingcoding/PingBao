PingBao

Windows开发环境 搭建指南
1. 安装zend studio
2. 安装zend server
3. 安装mysql

代码风格指南
1. 设置zend studio
Window->Preferences->General->Editors 选中Insert spaces for tabs
Window->Preferences->PHP->Code Style->Formatter, Edit...->Indentation, 设置Tab policy = Spaces , Indentation Size = 4
2. 目录名，文件名使用小写字符
3. 类名单词首字母大写
4. 属性名，方法名除首字母外，其余单词首字母大写
